TEMPLATE FORM LOGIN & SIGNUP SLIDING RED BLACK MINIMALIS
=====================================================

Terima kasih telah membeli template sliding 2-in-1 yang unik ini!

1. FILE YANG DISERTAKAN
-----------------------
* index.html: File utama (HTML, CSS, JavaScript).
* README.txt: Panduan penggunaan ini.

2. CARA MENGGUNAKAN
-------------------
a. Melihat Hasil:
   Cukup buka file 'index.html' di browser web apa pun (Chrome, Firefox, Edge).

b. Memodifikasi Teks/Link:
   Buka file 'index.html' menggunakan text editor (VS Code, Notepad++). Anda bisa langsung mengganti teks seperti "Login Form," "Email Address," atau link "Forgot password?".

3. PANDUAN MENGGANTI WARNA (MERAH)
----------------------------------
Template ini menggunakan skema warna Merah/Maroon. Untuk mengubahnya, Anda harus mengganti kode Hex pada bagian <style> di 'index.html':

1. Ganti Latar Belakang Utama:
   background: -webkit-linear-gradient(left, #660008,#df1111,#df1111, #660000);
   (Ganti semua kode Hex Merah ini dengan skema warna Anda sendiri).

2. Ganti Warna Tab Slider:
   background: -webkit-linear-gradient(left,#660000,#800000,#b30000, #e60000);

3. Ganti Warna Link:
   Ganti kode #e20c0c di bagian .pass-link a dan .signup-link a

4. CATATAN PENTING
------------------
* Ini adalah template front-end. Anda memerlukan back-end (PHP, Node.js, dll.) untuk memproses data input formulir.
